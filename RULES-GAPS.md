# RULES-GAPS

Open rules questions surfaced during implementation. Each of these is a spot
where the design docs are silent; the engine implements the most conservative
interpretation (behind a RulesConfig flag where applicable) and tags the code
with `// RULES-GAP:`. **Decide these with the designers, then delete the tag.**

Generated by `npm run gen:rules-gaps` — do not edit by hand.

1. **`packages/engine/src/cards.ts:58`** — Q/K say "discard a number card ... that value". Ace's value as a discard is unstated; the flip table treats A as a number card, so we count A = 1.
2. **`packages/engine/src/cards.ts:85`** — a Joker drawn during Polymerization has no defined blackjack value. Conservative reading: 0 (it is neither a number nor a face card).
3. **`packages/engine/src/poker.ts:103`** — partial banks (<5 cards) are compared "by high-card among themselves" — implemented as card-by-card rank comparison (pairs/flushes in a partial bank carry no weight); if one partial is a prefix-tie of a longer one, the longer bank wins (an extra card beats no card).
4. **`packages/engine/src/poker.ts:180`** — extended-kicker comparison when one bank simply has more cards left over is unspecified; an extra card beats no card.
5. **`packages/engine/src/reducer.ts:343`** — K debuff floor — power cannot go below 0; a 0-power monster stays on the field (YGO convention; docs silent on negative power).
6. **`packages/engine/src/reducer.ts:453`** — flipping the battle position of a face-down monster: face-down defense → face-up ATTACK (a position flip is defense↔attack), and the face-down→face-up transition triggers its flip effect.
7. **`packages/engine/src/reducer.ts:535`** — attacker destroyed / bounced / turned before the attack resolves — the attack fizzles and is spent (no re-declaration).
8. **`packages/engine/src/reducer.ts:542`** — a monster appeared in response to a direct attack — direct attacks are only legal against an empty board, so the attack fizzles.
9. **`packages/engine/src/reducer.ts:626`** — attacker power exactly equals defender power AND suit (mirror card into a wall): nothing is destroyed, no wall-punish (YGO convention).
10. **`packages/engine/src/reducer.ts:677`** — deck runs out mid-Polymerization. If nothing was ever dealt the effect fizzles; otherwise the caster is forced to STAND on the current total. Docs silent; deck-out clock is already ticking.
11. **`packages/engine/src/reducer.ts:854`** — the docs say flipping set→attack is "always allowed"; read as exempt from the once-per-turn position-change limit, NOT from the YGO rule that a monster cannot be flip-summoned the turn it was set.
12. **`packages/engine/src/types.ts:71`** — cards "removed" from a bank (bank-trigger removal, wall-punish) go to a removed-from-game zone, not the graveyard — YGO "remove" convention; grave would let Heart revive them, which seems unintended. Flag-free for M1.
